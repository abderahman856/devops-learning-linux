Leve 0-1 , Here what I've learnt from this level 
- 'cat' it displays the text and allows you to show the content of file


Leve 1-2, Here what I've learnt from this level 
-  Using './' to tell Linux that '-' is a filename, not a command flag.

Leve 2-3, Here what I've learnt from this level 
- Use 'Quotes' or 'Backslashes' (\ ) to handle spaces in filenames.


Leve 3-4, Here what I've learnt from this level 
- 'ls -a' reveals hidden files (those starting with a .)


Leve 4-5, Here what I've learnt from this level 
- The 'file' command identifies what kind of data is inside a file before you cat it.


Leve 5-6, Here what I've learnt from this level 
- 'find' with '-size' and '-readable' filters through thousands of files instantly.


Leve 6-7, Here what I've learnt from this level 
- 'find / -user bandit7 -group bandit6' locates files based on ownership.


Leve 7-8, Here what I've learnt from this level 
- 'grep' finds a specific needle (text) in a massive haystack (file).



Leve 8-9, Here what I've learnt from this level 
-'sort' and 'uniq -u' find the one unique line in a list of duplicates.



Leve 9-10, Here what I've learnt from this level 
- 'strings' extracts human-readable text from non-text (binary) files.



Leve 10-11, Here what I've learnt from this level 
- 'base64 -d' decodes data that has been obscured for transport.



Leve 11-12, Here what I've learnt from this level 
- 'tr 'A-Za-z' 'N-ZA-Mn-za-m'' shifts letters to bypass basic obfuscation.


Leve 12-13, Here what I've learnt from this level 
- 'xxd -r' reverses hex dumps, and 'file' tells you which decompression tool to use next.




Leve 13-14, Here what I've learnt from this level 
- 'ssh -i' uses a digital key file (identity) instead of a typed password.


Leve 14-15, Here what I've learnt from this level 
- nc (Netcat) sends data directly to a specific network port on a server.



Leve 15-16, Here what I've learnt from this level 
- 'openssl s_client' connects to services that are encrypted with SSL.


Leve 16-17, Here what I've learnt from this level 
- 'nmap' finds which services are listening, and 'openssl' retrieves keys from them.



Leve 17-18, Here what I've learnt from this level 
- 'diff' identifies exactly what changed between two versions of a file.



Leve 18-19, Here what I've learnt from this level 
- Adding a command to the 'ssh' string executes it before a "logout" script can run.



Leve 19-20, Here what I've learnt from this level 
- './binary' runs with the owner's permissions, allowing access to files you can't normally read.
