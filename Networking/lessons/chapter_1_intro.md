1. What is a Network?
Very simple definition

A network is a group of computers connected together so they can share data.

Technical view

A network allows:

  Sending messages
  
  Receiving messages
  
  Sharing resources

2. Why Computers Need Networking

Without networking

Each computer would:

  Work alone
  
  Store data locally
  
  Never communicate
  

3. Why networking exists

Computers need networking to:

  Share files
  
  Access websites
  
  Communicate with APIs
  
  Use cloud services
  
  Work as a system
