üõ†Ô∏è Assignment Challenges

Inside the assignment folder, you will find the following automation tools:

1. Basic Arithmetic Calculator (calculator.sh)

An interactive script that performs addition, subtraction, multiplication, and division.

Key Feature: Implements a safety check to handle and prevent division by zero errors.

2. File Operations Automator (bash_demo.sh)

A script that automates the creation of workspace environments.

Functionality: Creates a directory, generates a timestamped text file, and verifies the content immediately.

3. Permission & Existence Checker (file_checker.sh)

A DevOps tool used to verify file integrity and security settings.

Functionality: Checks if a file exists and provides a visual report (‚úì/‚úó) on Read, Write, and Execute permissions.

4. Smart Backup Utility (backup_script.sh)

A backup automation tool that organizes files into timestamped archives.

Functionality: Collects all .txt files from a source directory, moves them to a uniquely named folder, and provides a final file count report.

5. Bonus: System Monitor (system_monitor.sh)

A high-level system administration script for health checks.

Metrics Captured: CPU Load, Memory Usage (Total/Used/Free), Disk Space, and a list of the Top 5 memory-consuming processes.

Output: Generates a timestamped .log file for historical tracking.
